## 🚀 如何使用

### 1. 安装技能

```bash
npx skills add lennney/skills
```

### 2. 开始使用

在 Claude Code 中输入：
```
/gate
```

系统会引导你完成建站全流程！

### 3. 可用的技能

- **建站流程**：gate-discuss → gate-init → gate-frontend → gate-seo → gate-deploy
- **设计类**：ui-ux-pro-max, component-design, styling-tailwind
- **工程类**：code-review, diagnosing-bugs, tdd
- **AI 辅助**：ai-coding-workflow, ai-code-review, ai-debugging
- **产品类**：prd-development, acceptance-criteria, user-story

## 💡 使用技巧

1. **从 gate 开始** - 不确定用什么技能？输入 `/gate`，系统会引导你
2. **直接调用** - 知道要什么？直接输入技能名称，如 `/ui-ux-pro-max`
3. **组合使用** - 可以组合多个技能，如先 `/gate-discuss` 再 `/ui-ux-pro-max`

# Changelog

## [1.1.0] - 2026-06-24

### Added
- 完整的路由系统，覆盖所有 55 个技能
- 新增路由指南文档 `ROUTING-GUIDE.md`
- 更新 README，添加详细的路由系统说明

### Changed
- 重构路由系统，从单纯的建站流程扩展到全流程覆盖
- 更新技能分类，更清晰的组织结构

### Improved
- 更好的用户体验，从 `/gate` 可以路由到任何技能
- 更详细的使用说明和常见场景

## [1.0.0] - 2026-06-20

### Added
- 初始版本发布
- 55 个专业技能
- 建站流程路由系统
- 设计、工程、AI、产品、优化、讨论等技能分类

### Skills
- **建站流程** (5个): gate-discuss, gate-init, gate-frontend, gate-seo, gate-deploy
- **设计类** (11个): ui-ux-pro-max, component-design, styling-tailwind 等
- **工程类** (6个): code-review, diagnosing-bugs, tdd 等
- **AI 类** (6个): ai-coding-workflow, ai-code-review 等
- **产品类** (3个): prd-development, acceptance-criteria, user-story
- **优化类** (5个): gate-perf, gate-a11y, gate-animate 等
- **讨论类** (5个): gate-discuss, domain-modeling, handoff 等

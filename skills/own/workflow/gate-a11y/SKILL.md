Load `accessibility` skill. Check these 4 areas:

1. **键盘导航** → Tab 顺序正确？所有交互元素都可键盘操作？
2. **屏幕阅读器** → 图片有 alt？表单有 label？ARIA 属性正确？
3. **对比度** → 文字 ≥ 4.5:1
4. **焦点管理** → 弹窗锁定焦点、关闭后返回

Lighthouse Accessibility > 95 为目标。

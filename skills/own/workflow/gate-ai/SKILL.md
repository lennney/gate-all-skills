# Gate: AI 辅助

开发中需要 AI 帮忙？这里按场景分类。

## 你在做什么？

### 💻 写代码
> 加载 `ai-coding-workflow`

怎么和 AI 结对编程：写任务描述 → 让 AI 生成 → 审查差异 → 迭代修复 → 提交。

### 🔍 审查代码
> 加载 `ai-code-review`

让 AI 审查 PR：正确性 → 可维护性 → 性能 → 安全 → 无障碍。

### 🧪 写测试
> 加载 `ai-testing`

让 AI 生成单元测试/组件测试/E2E 测试。描述要测什么 → 审查生成的测试 → 修到通过。

### 🐛 调试
> 加载 `ai-debugging`

Bug 修不动？给 AI 完整上下文：复现步骤 + 期望/实际结果 + 错误栈 + 相关代码。

### 🔧 重构
> 加载 `ai-refactoring**

安全重构：明确范围 + 设好验证条件 → 让 AI 执行机械转换 → 验证没坏。

### 📝 Git 操作
> 加载 `ai-git-workflow`

让 AI 写 commit message、PR 描述、生成 changelog。
